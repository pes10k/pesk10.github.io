Command Line Basics

*
Everything in the current directory
a*
Everything in the current directory that starts with "a"
.
Where i am now
..
Directory above me

Browsing Files

cd
Change directory
cd ..
Change to the directory above me
ex: cd example
Change to the directory called example

Reading Files

cat
Show the contents of a file
ex: cat example.txt
Show the contents of example.txt

Searching for files

find
Search for a file by name
find a-directory -name "important.txt"
Look for a file called "important.txt" in the directory a-directory
find . -name "other.txt"
Look for a file called "other.txt" in the current directory

Searching in files

grep query *
Search for the word "query" in all the files in the current directory.
grep password secrets/*
Search for the word "password" in all the files in the secrets directory
grep -i ponies animals.txt
Search to see if the word "ponies"" or "Ponies" or "PoNiEs" etc is in the file animals.txt.

Security

passwd
Change the password for your user account
ssh-keygen -t ed25519
Generate an authentication key.
